centos7部署openldap

前言

为了公司统一认证。实验环境centos7.2 + ldap2.4

一、初始化环境

1. 拉取镜像
2. 启动镜像 这里比较必须加–privileged=true /usr/sbin/init 要不然无法使用systemctl命令 报错如括号内（Failed to get D-Bus connection: Operation not permitted）
3. 同步docker容器时间
4. 关闭selinux和防火墙

二、安装openldap

yum安装

检查是否安装成功

三、配置openldap

1. 配置openldap管理员密码

slappasswd -s 123456
备注：会生产一个{SSHA}xxx 之后会用到

2. 修改olcDatabase={2}hdb.ldif文件

vim /etc/openldap/slapd.d/cn\=config/olcDatabase\={2}hdb.ldif
