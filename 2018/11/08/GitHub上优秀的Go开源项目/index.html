GitHub上优秀的Go开源项目

Nov 8, 2018

docker

虚拟化平台，开源的应用容器引擎

  • https://github.com/docker
  • 50990 stars
  • 14846 forks

kubernetes

Google出品，用于调度和管理docker的开源容器管理系统，利用他，可以方便的管理你的docker实例

  • https://github.com/kubernetes/kubernetes
  • 43510 stars
  • 15134 forks

awesome-go

他是一个学习go的资料网站，属于著名的awesome系列，里面关于go的资源非常详细

  • https://github.com/avelino/awesome-go
  • 36800 stars
  • 4783 forks

gogs

一款基于git的代码托管系统，类似于github和gitlab，不过其小巧易用，功能强大，部署方便

  • https://github.com/gogs/gogs
  • 27754 stars
  • 3216 forks

syncthing

开源的文件同步系统,它使用了其独有的对等自由块交换协议,速度很快,据说可以替换BitTorrent Sync

  • https://github.com/syncthing/syncthing
  • 22668 stars
  • 1918 forks

hugo

一款极速的静态页面生成器，让你可以很快的搭建个人网站，提供了多套主题可供使用，并且可以自己定制

  • https://github.com/spf13/hugo
  • 30130 stars
  • 3512 forks

grafana

一款开源监控度量的看板系统，可以接Graphite,Elasticsearch,InfluxDB,zabbix,open-falcon等数据源，定制化很高。

  • https://github.com/grafana/grafana
  • 24730 stars
  • 4505 forks

etcd

一款分布式高可用的键值存储系统，主要用于共享配置和服务器发现。使用简单速度快安全。

  • https://github.com/etcd-io/etcd
  • 21084 stars
  • 4222 forks

influxdb

可伸缩的数据库，使用场景主要用来存储测量数据，事件点击以及其他等实时分析数据，用来做监控性能很不错。

  • https://github.com/influxdata/influxdb
  • 14736 stars
  • 2149 forks

caddy

快速的跨平台的http/2 web服务器

  • https://github.com/mholt/caddy
  • 19039 stars
  • 1467 forks

beego

国产开源的高性能web框架

  • https://github.com/astaxie/beego
  • 17642 stars
  • 3688 forks

martini

web框架

  • https://github.com/go-martini/martini
  • 10342 stars
  • 1068 forks

cayley

Google开源的图数据库，这是一个NoSql数据库，适合处理复杂的，但是结构化低的数据,适用于社交网络，推荐系统等。

  • https://github.com/cayleygraph/cayley
  • 11877 stars
  • 1047 forks

nsq

一款开源的实时的，分布式的消息中间件系统。类似MQ

  • https://github.com/nsqio/nsq
  • 13720 stars
  • 1793 forks

codis

Codis是一个分布式Redis解决方案,其实就是一个数据库代理，让你在使用Redis集群的时候，就像使用单机版的Redis是一样的，对开发者透明。

  • https://github.com/CodisLabs/codis
  • 8301 stars
  • 2011 forks

delve

一款go应用开发的调试工具

  • https://github.com/derekparker/delve
  • 9660 stars
  • 834 forks

cobra

cobra是一个命令行go库，可以让你创建非常强大的，现代的CLI命令行应用

  • https://github.com/spf13/cobra
  • 9398 stars
  • 807 forks

shadowsocks-go

go版本的shadowsocks,是一种基于Socks5的加密传输协议传输方式。

  • https://github.com/shadowsocks/shadowsocks-go
  • 5084 stars
  • 2963 forks

pholcus

Pholcus（幽灵蛛）是一款纯Go语言编写的支持分布式的高并发、重量级爬虫软件，定位于互联网数据采集，为具备一定Go或JS编程基础的人提供一个只需关注规则定制的功能强大的爬虫工具。

  • https://github.com/henrylee2cn/pholcus
  • 4693 stars
  • 1249 forks
