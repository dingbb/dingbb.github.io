常用包介绍

Go语言标准库里面使用频率比较高的一些包。熟悉这些包后，使用go语言开发常规程序会事半功倍。

  • fmt 实现了格式化的输入输出操作。其中fmt.Printf()和fmt.Println()是使用比较频繁的函数
  • io 实现了IO相关接口的实现，比对os中系统相关功能io的封装，比如读写文件等。
  • bufio 它在io的基础上提供了缓存功能。提供缓存功能之后可以比较方便的提供ReadLine之类的操作
  • strconv 提供了字符串与基本数据类型之间的转换
  • os 提供操作系统功能的相关接口。包括文件操作进程管理。信号和用户账号等
  • sync 提供了基本的同步原语。在多个goroutine访问共享资源的时候，需要使用sync中提供的锁机制。
  • flag 提供命令行参数的规则定义和传入参数解析的功能。绝大部分的命令行程序都需要用到这个包
  • encoding/json json目前广泛用作网络程序中的通信格式。本报提供了json的基本支持
  • http 强大而且易用的包，也是golang互联网语言最好的佐证。只需要数行代码。就可以实现一个爬虫或者web服务器。
