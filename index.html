前言

做运维相关工作已经4年，最近琢磨着建立一个自己的知识库，用博客的形式，记录下，自己工作中的点滴。
希望能在这个过程中，不断的总结，思考。也希望自己写的文章能帮助到大家。经过网上找资料。最终选择了
hexo与github的方式。
hexo是一款基于Node.js的静态博客框架,本文是再mac部署的。
希望能帮助到大家。

准备工作

安装node

  1. Node.js官网下载相应平台的最新版本

  2. 安装node,在终端输入以下命令

    brew install node

    如果没有安装brew请参考mac上安装brew

  3. 验证是否安装成功
    node -v

    出现版本号如 v9.3.0 代表安装成功。

    安装git

  4. 通过brew安装
    brew install git
